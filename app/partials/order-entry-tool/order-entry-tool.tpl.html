Order entry tool is responsible for collecting order information from user, displaying order parameters, Displaying a Confirmation Dialog and routing order to the book. It will be part of a tabular box see Thinkor swim example below. Other Tabs will display Working orders and Positions. User Can manually enter the order using this tool ( Chose the pair, expiration, strike, price etc... ) or user can click the bid/ask in the options display and order will be filled out automatically.
